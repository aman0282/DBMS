OUTER JOIN

Outer joins in SQL are used to retrieve data from multiple tables while also including unmatched rows from one or both tables in the result set.

TWO TYPES OF OUTER JOINJOIN:

1.LEFT OUTER JOIN JOIN: A left outer join returns all rows from the left table (the first table mentioned in the query), and the matched rows from the right table. If there are no matching rows in the right table, NULL values are returned for the columns from the right table.

SYNTAX:

SELECT columns FROM table1 LEFT OUTER JOIN table2 ON table1.column_name = table2.column_name;

2.RIGHT OUTER JOIN:A right outer join returns all rows from the right table (the second table mentioned in the query), and the matched rows from the left table. If there are no matching rows in the left table, NULL values are returned for the columns from the left table.

SYNTAX:

SELECT columns FROM table1 RIGHT OUTER JOIN table2 ON table1.column_name = table2.column_name;
