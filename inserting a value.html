INSERTING VALUES INTO A TABLE

*The INSERT statement is uesd to insert new rows or records into a table.

*Values can be inserted for the selected columns of the table.

SYTANX FOR INSERT STATEMENT IS:

INSERT INTO TABLE_NAME VALUES (VALUE1,VALUE2,VALU3,......VALUEN);
