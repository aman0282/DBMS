OFFSET - Clause

In MySQL, the OFFSET clause is used in conjunction with the LIMIT clause to specify the number of rows to skip before starting to return rows from the result set. It's commonly used for pagination, where you want to retrieve a subset of rows starting from a specific position in the result set.

SYNTAX :

SELECT column1, column2, ... FROM table_name LIMIT number_of_rows OFFSET offset_value;
