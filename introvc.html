VARIABLES IN PL/SQL?

In PL/SQL, variables are used to store and manipulate data within a program. They can hold various types of data, such as numbers, characters, dates, and more. Here's how you declare and use variables in PL/SQL:

SYTAX FOR VARIABLE DECLARATION:

variable_name datatype [NOT NULL := initial_value];

VARIABLE SCOPE:

Variables declared in the declaration section are visible within the block where they are declared. They can be referenced in subsequent PL/SQL statements within the same block.

INITIALIZING VARIABLES:

Variables can be initialized with an optional default value using the := assignment operator. If a variable is not initialized, it contains a NULL value by default.

SYNTAX FOR INITIALIZING VARIABLES:

variable_name := value;
