EQUI JOIN

Equi joins in SQL are a specific type of join where the condition for joining tables is based on the equality of values in specified columns.

TWO TYPES OF EQUI JOIN:

1.INNER EQUI JOIN: inner equi join returns only the rows that have matching values in both tables based on the specified join condition.

SYNTAX:

SELECT columns FROM table1 INNER JOIN table2 ON table1.column_name = table2.column_name;

OUTER EQUI JOIN: outer equi join returns all the rows from both tables, along with the matched rows based on the join condition.S

TWO TYPES OF OUTER JOIN:

SYNTAX:

SELECT columns FROM table1 LEFT OUTER JOIN table2 ON table1.column_name = table2.column_name;
OR
SELECT columns FROM table1 RIGHT OUTER JOIN table2 ON table1.column_name = table2.column_name;
