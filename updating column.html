Updating a column Of A Table:

*The Update command is used to update or modify the existing rows in a table.

*Updation can be carried out for all the rows in a table or selected rows.

*The Update statement modifies specific rows if the WHERE clause is specified.

SYTANX OF UPDATING A COLUMN :

Update table_name set column_name = value where [condition];
